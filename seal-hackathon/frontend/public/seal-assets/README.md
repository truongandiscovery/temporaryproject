# SEAL Hackathon Photo Assets

Place the real competition photos in this folder with these filenames:

- `seal-photo-1.jpg` - main landing hero photo
- `seal-photo-2.jpg` - login page and secondary landing photo
- `seal-banner-spring-2026.jpg` - register page and banner thumbnail

The app falls back to the SVG visuals in this folder when the JPG files are not present.
